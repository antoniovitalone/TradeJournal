## Packages
recharts | Powerful, declarative charting library for the performance curve
date-fns | Reliable date formatting for trades and charts
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes without style conflicts

## Notes
- App is designed with a strict "Dark Mode Finance/Trading" aesthetic.
- Forms use z.coerce for all numeric inputs to match API expectations.
- Requires standard Shadcn UI components (Card, Button, Input, Table, etc.) which are assumed present in the environment.
